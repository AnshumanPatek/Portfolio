# Anshuman Patek - Portfolio 2025

> Full Stack Developer building scalable systems that live and breathe in the cloud ☁️

A modern, animated portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## ✨ Features

- ⚡ **Next.js 14** - React framework with App Router
- 🎨 **Tailwind CSS** - Utility-first styling
- ✨ **Framer Motion** - Smooth animations
- 📱 **Fully Responsive** - Mobile & desktop optimized
- 🎯 **TypeScript** - Type-safe code
- 🌙 **Dark Theme** - Modern dark design

## 📦 Tech Stack

**Frontend:** React, Next.js, TypeScript, Tailwind CSS
**Animations:** Framer Motion
**Icons:** Lucide React
**Deployment:** Vercel (recommended)

## 🛠️ Project Structure

```
Portfolio-2025/

│   ├── Hero.tsx
│   ├── Skills.tsx
│   ├── Projects.tsx
│   ├── About.tsx
│   └── Contact.tsx
├── hooks/            # Custom React hooks
└── public/           # Static assets
```

## 🚀 Build & Deploy

```bash
# Build for production
npm run build

# Preview production build
npm start
```

### Deploy to Vercel

1. Push code to GitHub
2. Import on [vercel.com](https://vercel.com)
3. Deploy! ✨
